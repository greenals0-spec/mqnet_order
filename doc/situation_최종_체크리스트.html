3. 시스템 아키텍처 특이사항

이중화 동기화 엔진 Sync Engine

PersonalInfos 번들이 저장되는 즉시 RDBMS(Supabase/PostgreSQL) 내 users 테이블 계정 정보를 SHA-256 해시값 기반으로 자동 생성/동기화합니다. 이를 통해 지식창고 풀 기반 인증과 전통적인 RDBMS 인증이 완전한 정합성을 유지하며 상호 보조하도록 설계되었습니다.

지식창고 중복 데이터 제거 JSON Bypass

기존에 대용량 Employee 및 Attendance 번들이 JSON 지식창고 파일에 중복으로 누적되던 구조를 최적화했습니다. 백엔드의 업로드 API를 우회 설계하여, 실시간 DB 테이블에만 기록되게 함으로써 디스크 I/O 병목을 원천 해결하고 데이터 무결성을 향상시켰습니다.

Codeless System 아키텍처 구현 Codeless AI

AI 에이전트와 지능형 의사결정 모듈이 별도의 복잡한 코딩 변경 없이도 데이터 모델 및 규칙을 자율적(Codeless)으로 학습하고 운영할 수 있도록 AI 지능형 운영 시스템 논문 설계 사항을 시스템 전반에 성공적으로 정착시켰습니다.

본 결과서에 기재된 모든 검증 항목은 MQnet SaaS 플랫폼 및 스마트 오더 연동 규격에 따라 자동화 시뮬레이션을 통과하였으며, 계획서의 요구사항에 완벽히 부합함을 인증합니다.
