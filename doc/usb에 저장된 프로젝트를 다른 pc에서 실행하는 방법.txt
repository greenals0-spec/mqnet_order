usb에 저장된 프로젝트를 다른 pc에서 실행하려면 어떻게 해야하는지 알려줘

현재 코드가 모두 USB(D:\CodeLess)에 저장되어 있기 때문에, 집에 가셔서 USB를 꽂기만 하면 그대로 이어서 작업하실 수 있습니다!

다만, 집의 PC에서도 이 서버들을 실행하려면 파이썬(Python)과 Node.js라는 두 가지 실행 환경이 깔려 있어야 합니다.

집에 가셔서 다음 순서대로 진행해 주세요.

1단계: 필수 프로그램 확인 및 설치
집 PC에 다음 두 프로그램이 없다면 공식 홈페이지에서 다운로드하여 설치해 주세요.

Python: 파이썬 다운로드 (설치 시 하단의 'Add Python to PATH' 체크 필수)
Node.js: 노드JS 다운로드 (LTS 버전 권장)
2단계: 백엔드(AI 엔진) 실행하기
USB를 꽂고 CodeLess\situation-backend 폴더로 들어갑니다.
해당 폴더에서 주소창을 클릭하고 cmd를 입력해 터미널(명령 프롬프트)을 엽니다.
PC가 바뀌었으므로 패키지를 다시 설치해 주는 것이 안전합니다. 아래 명령어를 차례대로 복사해서 붙여넣고 엔터를 치세요.
python -m venv venv
venv\Scripts\activate
pip install fastapi uvicorn google-generativeai pydantic python-dotenv
설치가 끝나면 서버를 실행합니다.
uvicorn main:app --reload --port 8000
3단계: 프론트엔드(상황실 화면) 실행하기
다시 USB의 CodeLess\situation-room 폴더로 들어갑니다.
마찬가지로 주소창에 cmd를 쳐서 새로운 터미널을 엽니다. (백엔드 터미널은 켜두셔야 합니다!)
아래 명령어로 관련 패키지들을 설치합니다.
npm install
설치가 끝나면 화면 서버를 실행합니다.
npm run dev
이제 집 PC의 브라우저에서 http://localhost:5173/ 에 접속하시면, 지금 보시는 완벽한 똑같은 화면을 집에서도 띄워놓고 테스트해 보실 수 있습니다! (물론 Gemini API 키를 얻으셨다면 백엔드 폴더 안에 .env 파일을 만들고 넣는 것도 잊지 마세요!)
